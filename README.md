# TickTacToe in Flask
[![Python Version](https://img.shields.io/badge/Python-3.8-brightgreen.svg)](.)

A Flask app to play ticktactoe

## Installation
using git clone
```bash
git clone https://github.com/raseeksht/flaskticktactoe.git
cd flaskticktactoe
pip install -r requirements.txt
```

## Usage
```bash
python3 main.py
```
